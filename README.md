# Game of Life Kata

[Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) with an infinite board; test-driven designed in Kotlin with a focus on immutability and sequences. 

**Build**
```
./gradlew build
```

**Only Detekt**
```
./gradlew detekt
```

**Only Tests**
```
./gradlew test
```
